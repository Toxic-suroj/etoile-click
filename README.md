# Étoile Photographie — Premium Portfolio Website

A luxury photography portfolio website crafted for a professional photographer brand. The site includes full responsive sections, elegant dark styling, smooth interactions, and rich portfolio presentation.

## Included Files

- `index.html` — premium portfolio page with home, portfolio, gallery, about, services, pricing, testimonials, blog, and contact sections
- `styles.css` — premium dark mode styling, responsive layout, glassmorphism, and animation states
- `script.js` — theme toggle, mobile menu, portfolio filtering, lightbox preview, testimonial slider, contact form feedback, and reveal animations
- `404.html` — branded error page
- `sitemap.xml` — site map for SEO

## Usage

Open `index.html` in a browser to preview the website. The site is fully static and optimized for modern devices.
